Мои критерии удобной формы авторизации

  • При открытии формы автоматически перемещать фокус на поле ввода логина.
  • Использовать почту или логин для авторизации.
  • Дать возможность посмотреть пароль скрывающийся за звездочками.
  • Использовать модальные окна, они уменьшают ощущение продолжительной последовательности действий.
  • Использовать понятные надписи на кнопках, избегать абстрактных "далее" или "продолжить".
  • Использовать отдельные заголовки для полей, не размещать их в поле ввода.
  • Использовать клавиатуру для управления формой. Закрывать форму по нажатию "Esc"
  • Использование анимации для привлечения внимания, объяснения того что происходит и развлечения пользователя.
